REST Key-Value Store

Small key-value store built with Sinatra and Memcached.

Usage

Create / Update

POST /namespace/key
  • value is given as a parameter.
  • Using PUT to update is not supported, because Memcached does not support atomic get+set operation.
  • $ curl -d "value=looooong" kvs.api.enemy.fi/cats/longcat

Read

GET /namespace/key
  • If key is not found, then a HTTP 404 is returned.
  • $ curl kvs.api.enemy.fi/cats/longcat

Delete

DELETE /namespace/key
  • If key is not found, then a HTTP 404 is returned.
  • $ curl -X DELETE kvs.api.enemy.fi/cats/longcat
